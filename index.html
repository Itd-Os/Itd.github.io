Itd Os v0.8

Linux-Based for future laptops

Itd Os v0.8 is an operating system focused on privacy created to
make the Linux experience accessible to all with simple
documentation and apps suite designed to be user friendly
ssand read by minimizing the use of resources on the computer

Get it now Check our github repo

Secure & Private

Itd Os is a privacy-focused operating system with a built-in firewall
and a powerful antivirus that ensures greater security. We have also restricted the use of
services that require access to your data replacing Chrome and Google with Firefox and DuckduckGo

Fast and Light

Itd Os is light and fast thanks to the libraries that allow your apps to open faster
than other operating OS. Speaking of apps, we are working on a new suite of web apps
and AppImage that is fast, light and easy to install thanks to the integrated libraries
and web apps that use the HTML framework integrated in Itd Os to allow less use of your PC. resources.
